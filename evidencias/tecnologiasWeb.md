# Tecnologías Web

## Plataforma y desarrollo

- **WordPress** como sistema de gestión de contenidos (CMS).
- **HTML5** como estructura de marcado principal de páginas.
- **CSS personalizado** para estilos visuales adaptados.

## Complementos y plugins

El sitio incorpora plugins de WordPress personalizados o específicos para funciones como:

- Navegación mejorada.
- Directorio de profesionales médicos.
- Formularios de contacto y suscripción.

> **Nota:** La propia empresa que realizó el desarrollo menciona la integración de estos complementos adaptados al proyecto.
https://www.ptg.digital/en/proyectos-portfolio/hospitales-pascual-en/?utm_source=chatgpt.com

## Aspectos UX/UI

- **Sitio responsive** (adaptado para dispositivos móviles y escritorio) — implícito por el uso de WordPress y diseño moderno.
- **Menús multidioma** (español / inglés / portugués) visibles en la plantilla.

## Seguridad básica

- **HTTPS (SSL/TLS)** — esperado por defecto para portales sanitarios actuales.
